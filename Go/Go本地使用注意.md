## 1.win10下配置好环境变量

```
GOROOT:D:\Go

GOPATH:D:\gopath
```

## 2.go运行环境

go代码建立在`D:\gopath\src`，然后在里面创建不同的文件夹来安排自己的项目或者代码，同一个目录下不能存在多个`package`和`main`，不然会报错
